Tupperware

-My mother-in-law is district manager at Tupperware Hamburg, Germany and asked if there is a way to get an E-Mail based Newsletter page, which sounded perfect for me and my skillset. The page is coded with HTML, CSS and JS. Whenever someone submits to the newsletter his/her data is send directly into the mailbox of my mother-in-law so she can add the data directly into the distributor.-

-I redesigned the page some weeks ago, because I improved and learned more about alignment, negative margins and interactivity.-

-The page is fully written in German.-
